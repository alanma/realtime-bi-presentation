Data Driven Documents (D3)

Overview

  • Popular for Visualization (Maps!)
  • Join Semantics between DOM and Data
  • Low Level Package
  • Spawning High Level Add-Ons
    • NVD3 (Novus)
    • Rickshaw (Shutterstock)
    • DC.js (Nick Zhu)
    • Vega

Selections

  • d3.select for single nodes
  • d3.selectAll for multiple nodes
  • Joined to data using data
  • Chained style for most operations

Making a Selection


//Select all of the <li> elements inside of <ul id="selection1">
var ds  = ["Item 1","Item 2","Item 3","Item 4"];
var items = d3.select('ul#selection1').selectAll('li').data(ds);
